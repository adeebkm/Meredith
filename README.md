# Meredith Mueller - Google Search Simulation

A Google search simulation for **Meredith Mueller** with interactive LinkedIn and Facebook profiles.

## Features

- **Google Search Results**: 5 pages of realistic search results for "Meredith Mueller"
- **LinkedIn Profile**: Full interactive LinkedIn profile (click LinkedIn results to view)
  - Financial Analyst at Great Lakes Financial Services
  - University of Michigan education
  - Experience, skills, and activity posts
- **Facebook Profile**: Complete Facebook profile (click Facebook results to view)
  - Posts about hiking, brunch, and pets
  - Friends list and "People you may know"
  - About section with work and education

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to see the simulation.

## Meredith Mueller Profile

- **Job Title**: Financial Analyst
- **Company**: Great Lakes Financial Services
- **Education**: University of Michigan (BBA Finance, 2015-2019)
- **Location**: Chicago, Illinois
- **Experience**: ~5 years in financial analysis and investment research
